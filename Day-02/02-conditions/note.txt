1. if Statement
The if statement executes a block of code only if a specified condition is true.

Syntax:


if (condition) {
    // Code to execute if condition is true
}

2. if-else Statement
The if-else statement provides an alternative block of code if the condition is false.

Syntax:


if (condition) {
    // Code to execute if condition is true
} else {
    // Code to execute if condition is false
}

3. if-else if-else Statement
The if-else if-else statement allows multiple conditions to be checked in sequence.

Syntax:

if (condition1) {
    // Code to execute if condition1 is true
} else if (condition2) {
    // Code to execute if condition2 is true
} else {
    // Code to execute if all conditions are false
}


4. Nested if Statement
A nested if statement means an if statement inside another if statement.

Syntax:

if (condition1) {
    if (condition2) {
        // Code to execute if both condition1 and condition2 are true
    }
}

5. Ternary (Conditional) Operator
A shorter way to write an if-else statement.

Syntax:

condition ? expression1 : expression2;


6. Switch Statement
The switch statement is used when you need to compare a value with multiple possible cases.

Syntax:

switch (expression) {
    case value1:
        // Code to execute if expression === value1
        break;
    case value2:
        // Code to execute if expression === value2
        break;
    default:
        // Code to execute if no case matches
}
